发表于 2015-12-23   |  

终于开始使用hexo了，这也是一个机缘巧合，之前一直在找寻WP的替代品，搜着搜着就出现了hexo，然后就悄悄的部署了，过程中也遇到不少问题，现把实操留下，希望有所帮助。

安装过程

请参考：
http://ibruce.info/2013/11/22/hexo-your-blog/
我是按照这个教程往下做的，最好在结合一下官方的说明会更加可口美味。

遇到的问题

我相信很多人正常安装肯定没有问题，官方以及各种教程都说这是分分钟的事，也许是我RP确实不好，也比较污，所以安装中出现两个问题：

  1. hexo d 失败，查询之后得知需要修改github为git，同时做一个npm install hexo-deployer-git –save
  2. hexo d 还是失败，找了很久，显示
    Run git config --global user.email "you@example.com" git config --global user.name "Your Name"
  3. 把github的账号和名称输入回车就可以了，然后再输入一次hexo d 就会弹出账号和密码，输入你的github账号密码就OK了；
